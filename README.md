# Лабараторная работа №1

#### Задание на лабораторную работу

Необходимо оценить алгоритмическую сложность сортировок, реализованных в рамках практики №6,7

Исследуюмые алгоритмы:
- Сортировка "Расческа";
- Сортировка Шелла;
- Быстрая сортировка;
- Пирамидальная сортировка.
